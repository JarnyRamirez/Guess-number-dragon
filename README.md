# 🐉 Dragon Intuition - Number Guessing Game

A modern, arcade-style web application where users test their numerical intuition. This project was developed with a focus on **Glassmorphism** design and a **Mobile-First** approach, ensuring a seamless experience across all devices.

## 🚀 Professional Philosophy: Prevention over Correction
In this project, I implemented a strict data validation layer. Instead of correcting user errors after they occur, the system **prevents** them at the source by restricting input ranges in real-time, ensuring data integrity and a smooth UX.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure and data shielding.
* **CSS3:** Advanced styling with Glassmorphism effects, neon borders, and responsive design.
* **JavaScript (Vanilla):** DOM manipulation, game logic, and real-time input validation.

## ✨ Key Features
* **Glassmorphism Interface:** A high-end, translucent UI for a magical arcade feel.
* **Dragon Ranking System:** Dynamic scoring that rewards intuition with legendary ranks.
* **Input Shielding:** Real-time prevention of out-of-range numerical entries.
* **Fully Responsive:** Optimized for mobile, tablet, and desktop.
